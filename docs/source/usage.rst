設定ファイル
==================================================

ウェブカメラごとに設定ファイルを用意する。
サンプルとして :file:`kwconf.example` を作っておいたので、それをコピーして適宜修正を加える。


.. literalinclude:: ../../kwconf.example
   :language: bash
   :linenos:


[$WEBCAM]
--------------------------------------------------

ConfigParser で使うセクション名。
ウェブカメラの名前に設定しておけばよい。

name = $CAMNAME
--------------------------------------------------

カメラ毎に保存先を分けるために使う変数
全部同じにしても動くはずだけど、後で編集することを考えると異なる名前にしたほうがよい。


uri = $QW-ADDRESS/snapshot.jpg
--------------------------------------------------

ウェブカメラ内でJPEGファイルが保存されている場所。
TS-WLCAMシリーズの場合は「QwatchADDRESS」の部分を該当のIPアドレスに書き換えればOK。
メーカやシリーズごとに違うかもしれないので、適宜調べて置き換える。


user = $USER
--------------------------------------------------

ウェブカメラにログインするときのユーザ名

pass = $PASS
--------------------------------------------------

ウェブカメラにログインするときのパスワード


base = $WEBHOME/%(name)s
--------------------------------------------------

画像／動画を保存するディレクトリ名を設定する変数。
``%(name)s`` の部分は、上にある ``name`` で置換される。
EXPERIMENTSには各実験グループ名（とか用途）をいれるつもり。


log = %(base)s/qwlatest.log
--------------------------------------------------

ログファイル名を設定する変数。
``%(base)s`` は上の ``base`` で置換される。
