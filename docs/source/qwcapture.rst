==================================================
画像をキャプチャする : ``qwcapture``
==================================================

- confファイルを引数にして実行する

.. code-block:: bash

   $ ./qwcapture.py QWCONF.conf


- 複数のconfファイルを指定することもできる

.. code-block:: bash

   $ ./qwcapture.py QWCONF.conf QWCONF2.conf


オプションについて
==================================================

- いちおうwgetのオプションが使えるようになっている
- ヘルプを確認すること

.. code-block:: bash

   $ ./qwatch.py -h
