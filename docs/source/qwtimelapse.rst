==================================================
タイムラプス動画の作成 : ``qwtimelapse``
==================================================

- 日付とconfファイルを引数にして実行する
- 日付には、「 ``today`` 」「 ``yesterday`` 」「 ``YYYY/mm/dd`` の書式」が使える
- 日付は **１個** しか指定できない（confファイルは複数指定できる）

  - 基本的に毎日更新するため、複数日をまとめてやる必要が（とりあえず）ないと思うから

.. code-block:: bash

   $ ./qwtimelapse.py DATE QWCONF.conf
